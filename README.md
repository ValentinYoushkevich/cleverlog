# cleverlog

Cleverlog — веб-система учета рабочего времени сотрудников.

## Что делает система
- Ведет рабочие логи по проектам (Work Log) с обязательными `Task Number` и комментарием.
- Учитывает отсутствия: `Vacation`, `Sick Leave`, `Day Off`.
- Считает факт часов за месяц, сравнивает с нормой и показывает отклонения.
- Формирует отчеты (по пользователю, по проекту, свод по месяцу) и экспорт в Excel.
- Дает администратору дашборд для контроля недоработки, переработки и незаполненных дней.

## Роли
- `User`: работает только со своими записями и отчетами.
- `Admin`: полный доступ к пользователям, проектам, календарю, нормам, закрытию месяца, аудиту.

## Ключевые правила (MVP)
- Нельзя логировать в будущие даты.
- Нельзя совмещать `Work Log` и `Absence` в один день.
- В проекты со статусами `On Hold` и `Closed` логирование запрещено.
- После закрытия месяца записи для `User` становятся read-only.

## Технологический стек
- Frontend: `Vue 3`, `Tailwind CSS`, `PrimeVue`.
- Backend: `Node.js`, `Express`, `PostgreSQL`.
- Инфраструктура: `Docker`.
